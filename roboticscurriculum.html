Robotics Curriculum Outline

This is a complete outline of the robotics curriculum. It is a comprehensive guide to building, programming, and controlling robots using LEGO EV3 and RobotC.

Introduction to Robotics

This lesson introduces students to the core concepts of robotics, the "sense, think, act" process, and the Robot Virtual Worlds platform as a safe way to get started.

Suggested Activities

  • ✓
    Build a Basic Robot: Students will follow instructions to build a simple two-wheeled robot with a single sensor.

Drivetrain & Movement

Students learn about the robot's drivetrain and how to make it move. The lesson covers the `motor[]` and `wait1Msec()` commands to control speed, direction, and duration of movement.

Suggested Activities

  • ✓
    The Straight Shot: Program the robot to drive forward for a set time and then stop.
  • ✓
    Square Dance: Program the robot to drive in a perfect square, reinforcing the need for precise timing.
  • ✓
    The U-Turn: Program the robot to make a 180-degree turn.

Programming Structure

This lesson focuses on the foundational rules of writing clean, readable code, including proper use of whitespace, comments, semicolons, and paired punctuation like `()`, `{}`, and `[]`.

Suggested Activities

  • ✓
    Code Cleanup: Correct a poorly formatted and un-commented piece of code to improve readability.

Variables & Data Storage

Students are introduced to variables as containers for data. The lesson covers different data types, including `int`, `float`, `double`, `string`, and `boolean`, and shows how to use them to store information.

Suggested Activities

  • ✓
    Speed Control Interface: Use a variable to store the robot's motor speed and control it with buttons on the EV3 brick.

Touch Sensors

This lesson introduces the touch sensor as a simple **digital sensor** with two states. Students learn how to read a `0` or `1` value to trigger actions based on physical contact.

Suggested Activities

  • ✓
    The Bumper Bot: Program the robot to drive forward until its touch sensor is pressed, then back up and turn before driving forward again.

Ultrasonic Sensors

Students learn about the ultrasonic sensor as an **analog sensor**. They will see how it uses echolocation to measure distance and how to use the `SensorValue[]` command to read its data in both metric and imperial units.

Suggested Activities

  • ✓
    The Wall Follower: Program the robot to drive until it detects a wall at a specific distance and then stop.

Control Statements

This lesson gives students the tools to make their programs more dynamic. It covers conditional statements (`if`, `else`, `else if`) for making decisions and repetitive statements (`while`, `for`) for repeating tasks.

Suggested Activities

  • ✓
    The Simple Maze Runner: Combine loops and conditional statements to program a robot to navigate a simple maze autonomously.

Putting it All Together

This final lesson is a capstone challenge that requires students to apply all the concepts they've learned throughout the course to solve a more complex problem.

Suggested Activities

  • ✓
    The Grand Challenge: Program the robot to navigate a complex obstacle course, complete a specific task, or compete in a timed challenge with other robots.
